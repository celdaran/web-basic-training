CSS

CSS stands for Cascading Style Sheets and it plays a big part in what we call the "presentation layer" of an interface.

HTML provides the structure and semantics of a document. CSS makes it pwetty.

Styles

A "style" is the name given to a set of attributes that describe the appearence of an element.

For example, here is a paragraph:

Far far away, behind the word mountains, far from the countries Vokalia and Consonantia, there live the blind texts. Separated they live in Bookmarksgrove right at the coast of the Semantics, a large language ocean. A small river named Duden flows by their place and supplies it with the necessary regelialia.

But let's say we want all of our paragraphs to have a certain style to them. Like the typeface, size, weight, color, background, indents, and so on. Well, we can group all of these attributes into a paragraph style and have it render like this:

Far far away, behind the word mountains, far from the countries Vokalia and Consonantia, there live the blind texts. Separated they live in Bookmarksgrove right at the coast of the Semantics, a large language ocean. A small river named Duden flows by their place and supplies it with the necessary regelialia.

Sheet

A "sheet" is just the term given to a collection of styles.
